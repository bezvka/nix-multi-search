**nix-multi-search** — это Python-скрипт для быстрого и параллельного поиска пакетов в различных каналах NixOS (от старых релизов до `unstable` и `master`).

Плюсы использование:
* **Параллельный поиск:** Использует `ThreadPoolExecutor` для одновременного опроса всех выбранных каналов.
* **Поддержка JSON:** Работает через `nix search --json` для получения точных данных.
* **Интерактивность:** Возможность выбора конкретных репозиториев для поиска.
* **Цветной вывод:** Удобное отображение версий и описаний пакетов.
* **Unfree Support:** Автоматически учитывает `NIXPKGS_ALLOW_UNFREE`.


Почему я сделал этот проект?

Идея родилась из практической необходимости. Мне нужно было установить hiddify, но пакета не оказалось ни в репозиториях версии 25.11, ни в unstable.

Стандартный поиск через search.nixos.org заставляет переключаться между версиями вручную, чтобы проверить наличие пакета в разных ветках. Это отнимало время. Я решил написать инструмент, который ищет пакет сразу по многим версиям репозиториев одновременно.

Изначально делал для себя, чтобы упростить жизнь, но понял, что это может быть полезно и другим пользователям NixOS.
